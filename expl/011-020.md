## (20) 二次方程式の解

入力 3（c）: 0 より大きく 100 以下 の実数<br>

```c
//sqet()のためにmath.hをinclude
//判別式を用いて場合わけするだけ

#include<math.h>

    float a, b, c;
    scanf("%f%f%f", &a, &b, &c);

    if (d>0) {
        printf("解は%.3fと%.3f", (-b+sqrt(d))/(2*a), (-b-sqrt(d))/(2*a));
    } else if (d<0) {
```
